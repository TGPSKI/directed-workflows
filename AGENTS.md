# Directed Workflows -- Agent Instructions

The canonical instructions for this project live in `SKILL.md` in this directory. Read that file for full instructions on how to create directed workflows.

## For Cursor Users

This repository can be installed as a Cursor skill for global availability:

```bash
git clone https://github.com/tgpski/directed-workflows.git ~/.cursor/skills/directed-workflows
```

Once installed, the skill is auto-discovered in every workspace. See `SKILL.md` for details.

## For Other AI IDEs

If you are an AI agent running in Claude Code, Windsurf, or another IDE that reads `AGENTS.md`, read `SKILL.md` in this same directory. It contains all conventions, templates, examples, and step-by-step instructions for generating directed workflows in the user's repository.
